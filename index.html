4 使用流程
①
【可选·方案C预备】提前抓取大麦 Cookie
若你的演出会显示「该渠道不支持购票」，需提前准备好 Cookie。
App Store 安装 Stream（免费）→ 开启抓包 → 打开大麦App进入该演出购票页 → Stream 找到 damai.cn 请求 → Request → Headers → 复制 Cookie 字段的完整值，备用。
②
Safari 打开大麦演出页并登录
进入该演出页面（即使显示渠道拦截也没关系），确认已登录账号
③
点击书签「大麦抢票」
脚本加载后自动检测渠道状态：
• 页面正常 → 直接进入倒计时 ✓
• 检测到拦截 → 自动尝试方案A（UA伪造+刷新）
• 方案A失败 → 面板显示方案C引导，粘贴 Cookie 注入
④
倒计时进行中，保持页面前台
面板显示配置信息和倒计时。不要切到后台或锁屏，否则 Safari 会暂停 JS 执行。
⑤
时间到 → 自动执行购票序列
自动：选票价档位 → 点立即购买 → 选观演人 → 确认
失败自动重试，最多 100 次。跳转到支付页后手动完成支付宝付款。
